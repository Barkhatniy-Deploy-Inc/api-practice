# Руководство студента по работе с API «Зебра-стат»

Добро пожаловать в образовательную среду «Зебра-стат» (Аналитика ДТП). Этот документ является основной инструкцией для проведения исследований в рамках учебного курса.


---

## 🔐 1. Группа: Авторизация (Authentication)
*С\u00A0этого раздела начинается ваша работа с\u00A0сервером.*

### `GET /api/v1/auth/me`
Позволяет убедиться, что\u00A0ваш ключ активен и\u00A0узнать, на\u00A0чьё имя он зарегистрирован.
*   **Заголовок:** `X-API-KEY: <ваш_ключ>`
*   **Зачем это нужно:** проверка работоспособности системы перед\u00A0началом больших запросов.

---

## 📚 2. Группа: Справочники (Dictionaries)
*Используйте эти данные для\u00A0декодирования кодов в\u00A0читаемые названия.*

### `GET /api/v1/dictionaries/regions`
Полный список субъектов РФ.
*   **Ответ:** массив объектов с\u00A0полями `region_code` (нужен для\u00A0фильтрации ДТП) и\u00A0`region_name`.

### `GET /api/v1/dictionaries/weather`
Справочник погодных условий.
*   **Зачем это нужно:** чтобы понимать, что\u00A0означает `weather_id=1` в\u00A0карточке ДТП (например, «Ясно»).

---

## 🚗 3. Группа: Реестр ДТП (Accidents)
*Доступ к\u00A0«сырым» данным о\u00A0происшествиях.*

### `GET /api/v1/accidents/`
Список всех происшествий с\u00A0мощной фильтрацией.
*   **Важные параметры:**
    *   `region_code`: фильтр по\u00A0региону (например, `1114`).
    *   `has_children`: только ДТП с\u00A0участием несовершеннолетних.
    *   `limit`: количество записей (макс. 100).
*   **Совет:** используйте пагинацию через\u00A0`offset` для\u00A0выгрузки больших наборов данных.

### `GET /api/v1/accidents/{id}`
Детальная информация о\u00A0конкретном ДТП.
*   **Что внутри:** полный состав участников (возраст, стаж, роль), данные о\u00A0транспортных средствах и\u00A0состоянии дороги.

---

## 📊 4. Группа: Аналитика (Analytics)
*Инструменты для\u00A0написания выводов в\u00A0отчетах.*

### `GET /api/v1/analytics/trends/prediction`
Прогноз аварийности на\u00A0будущий месяц.
*   **Математика:** линейная регрессия на\u00A0основе последних 12 месяцев.
*   **Как использовать:** сравните прогноз с\u00A0реальными данными в\u00A0следующем семестре.

### `GET /api/v1/analytics/reports/participant-profile`
«Портрет типичного виновника».
*   **Что вы получите:** статистическую моду по\u00A0полу, возрасту и\u00A0стажу вождения.
*   **Пример вывода:** «Мужчина, 26–45 лет, стаж более 10 лет».

## 🚀 Как выполнить первый запрос?
Если вы используете Python:
```python
import requests
headers = {"X-API-KEY": "ваш_секретный_ключ"}
response = requests.get("http://127.0.0.1:8000/api/v1/analytics/summary", headers=headers)
print(response.json())
```
